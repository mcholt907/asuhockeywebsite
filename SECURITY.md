# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability in the ASU Hockey Fan Website, please **DO NOT** open a public issue. Instead, please report it responsibly by emailing the project maintainer at mcholthaus@gmail.com with details about the vulnerability.

Please include the following in your report:
* A clear description of the vulnerability
* Steps to reproduce (if applicable)
* Potential impact of the vulnerability
* Your suggested fix (if you have one)

## Response Process
We will acknowledge receipt of your vulnerability report within 48 hours and will provide a more detailed response within 5 business days. We appreciate your patience and cooperation as we work to address any security issues.

## Disclosure Policy
Once a vulnerability is fixed, we will:
1. Notify the reporter
2. Release a patched version
3. Publicly acknowledge the fix in our release notes (unless the reporter prefers anonymity)

## Security Best Practices
When using the ASU Hockey Fan Website:
* Keep your browser and plugins updated
* Use strong, unique passwords
* Be cautious with personal information shared on the site
* Report any suspicious activity to the project maintainers

Thank you for helping keep our project secure!
